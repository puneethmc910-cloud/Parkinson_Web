Parkinson’s Monitoring with ESP32 + Sensors

Track tremor, gait, and balance using MPU6050 IMU, FSR pressure sensors, and flex sensors on ESP32. Connect in real time, visualize left vs. right leg pressure, and flag instability early.

Open Dashboard How it works

Real-time streaming

WebSocket-based feed from ESP32; auto-reconnect and optional demo mode.

Pressure symmetry

Left/right leg FSRs graphed live to spot stance asymmetry and freezing cues.

Tremor metrics

Accel magnitude to estimate tremor amplitude; add FFT later if needed.

ESP32 setup (HTTP/WebSocket)

Flash ESP32 with Wi‑Fi + WebSocket on port 81. Send JSON fields: t, ax, ay, az, gx, gy, gz, fsrL, fsrR, flex, part. Put the same IP in the Dashboard WebSocket field. The site will connect on LAN or AP mode (e.g., 192.168.4.1).

Privacy first

Processing stays in-browser; no cloud dependency. Use body-part tags, avoid personal identifiers, secure your Wi‑Fi.
