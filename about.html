About the Project

A lightweight, local-first toolkit to monitor Parkinson’s symptoms—tremor, gait, and balance—using an ESP32 with common sensors.

Sensors

MPU6050 (accel + gyro) for tremor and motion; FSR 402 / FlexiForce A201 for left/right foot pressure; Flex sensor for joint bend; optional MAX30100 for HR/SpO2.

Placement

IMU on shin or wrist; FSR pads under heel/toe or in insoles; flex across knee or finger.

How data flows

ESP32 samples sensors, sends JSON via WebSocket on port 81: {t, ax, ay, az, gx, gy, gz, fsrL, fsrR, flex, part}. The dashboard consumes this stream and renders live charts.

Privacy & safety

Runs locally; no cloud required. Use WPA2/WPA3, avoid personal identifiers, and clear logs if you store data.

ESP32 + browser = fast, local insights. Swap demo data for your live feed anytime.
